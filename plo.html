Virtual PL/0

Access this Project

The virtual PL/0 project emulates the basic functionality of the PL/0 language in C. It reads in PL/0 code from a text file, scans it into distinct symbols, and then compiles those symbols into machine code. It also runs the machine code within the emulator. The compiler uses a na√Øve, greedy implementation, with no boost to efficiency. The machine code emulator simulates a stack-and-register architecture to run the code.
