Programming Team Hackpack

Access this Project

The programming Team Hackpack is a collection of algorithms that can be printed out to assist in programming competitions. Code is designed for ease of printing and for rapid re-typing, with comments added to insure that on-the-fly modifications are as accurate as possible. The hackpack has one class for each major algorithm such as flow, and compilation classes for collections of minor algorithms like combinatorics (permutations, combinations, etc.)
